## Setup/Installation Requirements

1. Open Git Bash if on Windows and terminal if on Mac
2. Run the command

    ``git clone https://github.com/leark/currency-exchanger.git``

3. Run the command

    ``cd currency-exchanger``

4. Run the command

    ``npm init -y``

5. Run the command

    ``npm install``

6. Run the command

    ``npm run start``
